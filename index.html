Como atualizar a escala

Você pode alterar pelo próprio app na aba Administração, usando a senha cadastrada.

As alterações feitas no app ficam salvas no navegador do aparelho. Para usar em outro celular ou PC, use Exportar dados e depois Importar backup.

Também é possível editar diretamente o arquivo dados.js no GitHub e clicar em Commit changes.

Para não expor dados internos, use nomes abreviados ou códigos se o repositório for público.
